Title: [2019-01-14] Challenge #372 [Easy] Perfectly balanced

Given a string containing only the characters `x` and `y`, find whether there are the same number of `x`s and `y`s.

    balanced("xxxyyy") =&gt; true
    balanced("yyyxxx") =&gt; true
    balanced("xxxyyyy") =&gt; false
    balanced("yyxyxxyxxyyyyxxxyxyx") =&gt; true
    balanced("xyxxxxyyyxyxxyxxyy") =&gt; false
    balanced("") =&gt; true
    balanced("x") =&gt; false

## Optional bonus

Given a string containing only lowercase letters, find whether every letter that appears in the string appears the same number of times. Don't forget to handle the empty string (`""`) correctly!

    balanced_bonus("xxxyyyzzz") =&gt; true
    balanced_bonus("abccbaabccba") =&gt; true
    balanced_bonus("xxxyyyzzzz") =&gt; false
    balanced_bonus("abcdefghijklmnopqrstuvwxyz") =&gt; true
    balanced_bonus("pqq") =&gt; false
    balanced_bonus("fdedfdeffeddefeeeefddf") =&gt; false
    balanced_bonus("www") =&gt; true
    balanced_bonus("x") =&gt; true
    balanced_bonus("") =&gt; true

Note that `balanced_bonus` behaves differently than `balanced` for a few inputs, e.g. `"x"`.
